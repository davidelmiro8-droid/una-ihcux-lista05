No meu programa, usei algumas boas práticas de usabilidade para deixar tudo mais fácil e agradável de usar. Mostro o que está acontecendo com mensagens como “conectando” e “calculando”, evitando que o usuário ache que travou. Também tratei erros com try-catch, exibindo avisos claros quando algo dá errado. Além disso, organizei bem a interface, com cores e separadores, deixando tudo mais limpo e profissional. No fim, a aplicação ficou simples, mas bem mais intuitiva e confortável de usar.
